# 📖 Planner Literário

> Organize, registre e acompanhe sua jornada de leitura de forma simples e visual.

---

## 📌 Sobre o Projeto

O **Planner Literário** é uma aplicação web desenvolvida para auxiliar leitores a organizarem suas leituras, metas e registros literários em um só lugar.

⚠️ **Importante:**  
Este projeto foi desenvolvido **exclusivamente para fins de estudo**, com o objetivo de praticar conceitos de front-end e construção de interfaces interativas.

---

## 🎯 Objetivos de Aprendizado

- Estruturação de páginas com **HTML5 semântico**  
- Estilização com **CSS3**  
- Organização de layout e hierarquia visual  
- Criação de interfaces voltadas à experiência do usuário  
- Simulação de um sistema com utilidade prática  

---

## 🧭 Funcionalidades

- 📚 Registro de leituras  
- 🎯 Definição de metas literárias  
- 📝 Organização de informações sobre livros  
- 📊 Acompanhamento do progresso de leitura  
- 📱 Layout responsivo  

Planners literários costumam incluir elementos como listas de livros, metas e registros de leitura para acompanhar a evolução do leitor ao longo do tempo :contentReference[oaicite:0]{index=0}.

## 🧱 Estrutura da Aplicação

A aplicação foi organizada para simular um planner completo:

- 📌 Área de planejamento  
- 📖 Registro de livros lidos  
- ⭐ Avaliações e observações  
- 🧾 Listas (desejados, lidos, favoritos)  
- 📅 Organização por períodos ou metas
